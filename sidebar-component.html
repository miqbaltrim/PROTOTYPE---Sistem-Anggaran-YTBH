Sidebar Component

Copy blok <!-- SIDEBAR --> beserta CSS & JS-nya ke setiap halaman. Tambahkan data-page="nama" pada <body> agar item aktif otomatis ter-highlight.

<!-- 1. Tambah di <head> --> <link rel="stylesheet" href="sidebar.css"> <!-- 2. Set halaman aktif di <body> --> <body data-page="dashboard"> <!-- 3. Tambah main wrapper --> <div class="main"> <!-- konten halaman --> </div> <!-- 4. Tambah script --> <script src="sidebar.js"></script>
